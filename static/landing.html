QRaven

Lightweight event, ticketing and payments API with authentication and notifications.

What it does

QRaven provides backend services to manage events, sell tickets, accept payments, and send notifications. It exposes a REST API for common operations and includes metrics and Swagger documentation.

  • Authentication & access tokens
  • Create, list and manage events
  • Ticket reservations and issuance
  • Payment processing integration points
  • Notifications and metrics for observability
API base: /api/v1  •  Docs: Swagger UI (or /swagger)
